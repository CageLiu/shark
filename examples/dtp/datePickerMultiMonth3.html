jquery.datePicker multi month plugin plugin example: showing how you can use the datePickerMultiMonth plugin

< date picker home

This page illustrates the use of the "plugin plugin" datePickerMultiMonth. It is a plugin which builds on the functionality of the date picker to allow you to have datePickers which display multiple months at a time. The construction of the multimonth plugin illustrates the flexiblility of the datePicker and how you can add functionality without increasing the filesize of the base plugin.

Note that the relevant events are passed through to this page. For dpMonthChanged the month and year passed correspond to the first month out of the ones shown.

You can also call dpmmGetSelected on the datePickerMultiMonth which is the same as dpGetSelected on a normal date picker. Example.
