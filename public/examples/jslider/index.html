jSlider插件demo演示：


jSlider插件参数说明表
参数名 参数值(默认值可以省略)  
effect dirY（默认）：垂直滚动  
dirX：水平滚动
Txt：文本垂直滚动
Fade：淡出
Loop：水平左右点击滚动
autoPlay true（默认）： or false  
speed 动画速度时间，默认"normal"，可以使用毫秒或者JQ中的fast，slow，normal  
timer 滚动间隔时间，默认"1000，可以使用毫秒或者JQ中的fast，slow，normal  
claNav jslider_handler（默认）：触点对象数组父级  
claCon jslider_content（默认）：滚动对象或滚动对象父级  
steps 1（默认）：滚动几个  


滚动一：

$("#temp1").jSlider({
    effect : "scrollY",
    speed : "normal",
    timer : 3000
});

滚动二：

$("#temp2").jSlider({
    effect : "scrollX",
    speed : "normal",
    timer : 3000
});

滚动三：

$("#temp3").jSlider({
	effect : "fade",
    speed : "normal",
    timer : 3000
});

滚动四：

‹ › 
$("#temp4").jSlider({
